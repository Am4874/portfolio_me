## Installation

To set up the portfolio website locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone 
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio_me
   ```

3. Open the `index.html` file in your preferred web browser.
